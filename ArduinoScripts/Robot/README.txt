Immer an den Upload Switch denken!

Falls es mal prbleme bei dem Kommunikation gibt den Reset button drücken, das hilft!

Server starten: python -m http.server 8080

im Browser: http://localhost:8080/GUI.html

ESP bekommt die verarbeiteten Befehle vom Server (PC), das bedeutet entweder ein Bild wird gesendet oder der Befehl wird an den Arduino weitergegeben.

Arduino führt entsprechend des Befehls die Bewegung aus!
